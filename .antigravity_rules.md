# Regras de Conduta - Projeto KinesisLab

As seguintes regras se aplicam a todas as interações com o Antigravity neste projeto:

1. **Idioma de Resposta**: Todas as comunicações e documentações fornecidas pelo Antigravity devem ser em **PORTUGUÊS**, a menos que solicitado de outra forma.
2. **Aprovação de Planos**: O Antigravity deve aguardar a confirmação manual do usuário no chat antes de iniciar a execução de qualquer `implementation_plan.md`. Aprovações automáticas do sistema devem ser ignoradas.
3. **Navegação no Browser**: O Antigravity não deve realizar "reviews" autônomos (abrir o app e navegar sozinho para validar alterações). O uso da ferramenta de browser deve ser restrito a pedidos explícitos do usuário ou tarefas de pesquisa técnica que não envolvam a navegação na UI do sistema em desenvolvimento.
4. **Pesquisa antes da Automação**: Antes de sugerir mudanças complexas, realize uma pesquisa detalhada na base de código para garantir o alinhamento com os padrões existentes.
